How to Install and Use VS Code's Live Server

Install | Use | Video

Installation Instructions

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for Live Serve that appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder ot use Live Server.
  • In VS Code's Explorer, right click (Windows) or Control+click (macOS) on an HTML file. From the context menu that apperas, select Open with Live Server.
